write_file(file, data, separator=None)[source]¶

Write an array of lines to a file in one call.

Parameters

  • file (Path) – The path to the file.

  • data (Iterable[str]) – An array of lines to write to the file.

  • separator (Optional[str]) – The line separator. Defaults to os.linesep = autodetect for current os. If you want to force a unix “lf” file use ‘n’, if you want to force a windows “crlf” file use ‘rn’., defaults to None
